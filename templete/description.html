This project aims to develop a component of comfort for the end user by allow- ing him the luxury to monitor his power consumption in real time. This will be beneficial for the customer and will grant him an economical advantage. This will be carried out via use of Wi-Fi and IoT with the end result displayed on a website or an app. Additionally, this project will inform the end user regarding any over voltage or under voltage condition that can be very dangerous for sen- sitive electronic devices. This will be carried out via conditional programming. Lastly, we aim to develop a sub part of a de-centralized home automation sys- tem in the form of a smart AC socket that will communicate real time data of particular devices that the socket has been attached to.
